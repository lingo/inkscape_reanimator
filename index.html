Inkscape reanimator

Preview (perhaps one day edit) animations created in Inkscape as 1-frame-per-layer. Loads the Inkscape SVG file and animates by changing layer visibility. Watches for changes to the file loaded so that your changes made in Inkscape are instantly animated.
